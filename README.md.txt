# 📊 Customer Segmentation Using K-Means Clustering

## Overview

SegmentVista is an interactive Customer Segmentation Dashboard that uses the K-Means Clustering algorithm to group customers based on purchasing behavior and demographic patterns.

The dashboard allows users to dynamically select the number of clusters and visualize customer groups through interactive charts and analytics.

---

## Features

### 🎯 Customer Segmentation
- K-Means Clustering
- Dynamic Cluster Selection
- Interactive Cluster Visualization

### 📈 Data Analytics
- Customer Distribution Analysis
- Cluster Statistics
- Comparative Insights

### 🎨 Modern Dashboard
- Responsive Design
- Interactive Charts
- User-Friendly Interface
